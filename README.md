# **These documents are UNOFFICIAL!!!!**

This repository contains documents that may be useful for certain key volunteers of the FIRST Robotics Competition 2026 game, Rebuilt.

The rule cheat sheet uses most of the summary text, but some of them have been reworded to make it more useful.

Summary of contents:

| File | Description |
| --- | --- |
| rebuilt_frc_rule_cheatsheet.pdf/rebuilt_rfc_rule_cheatsheet.pptx | Quick cheat sheet of important rules and their penalties. |
| violation_sheet.pdf | A form that can be handed between Key Volunteers to track major violations. Recommended for lamination and used with dry erase markers.|
